What I am doing now

Vipassana

I attended a 10-day course in Vipassana meditation and found it to be a transformational experience. I wrote about the experience here. I now sustain this practice by meditating twice a day.

Willow DB

My next project is to build a simple web tool that will help me stay in touch with people. If I have fallen out of touch with somebody, the tool will remind me to get back in touch. I have already built a working prototype that with Appsmith, a low-code solution to build simple frontend apps. I intend to rewrite the front-end in vanilla HTML and Javascript before sharing the tool for wider use.

Karate

At work, we have migrated to Karate DSL, for API Testing and have found it to be a powerful tool that is a pleasure to work with. Our focus is to now harness this tool's ability to perform Contract Testing by using API test doubles.

Frontend Testing

At work, we are also exploring various frontend test frameworks to replace our existing setup. We have zeroed in Cypress and Playwright, both of which I find to be extremely well crafted, thereby making it a difficult choice.

Sport

I have started playing some friendly football here in Berlin (admittedly 'Lazy and Lousy' football). I look forward to it every week!

I continue to sustain 2 yoga sessions a week. It serves as a great complement to my efforts with Vipassana meditation.
